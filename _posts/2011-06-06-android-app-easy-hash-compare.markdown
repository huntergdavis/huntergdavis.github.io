Easy Hash Compare is an easy to use app to compare the MD5 hash values of two files on your Android file system. It will allow you to know, with some certainty, that two files are equal.

[![](/content/images/2011/06/device22-180x300.png "Easy Hash Compare 2")](/content/images/2011/06/device22.png)

You can see the permalink page [here](/android-app-easy-hash-compare), and download it from the Android app store [here](https://market.android.com/details?id=com.hunterdavis.easyhashcompare).
